1. User Stories:
- As a user I want to go to website and get random jokes
- As a user I want a counter where I can see how many jokes I already laughed about
- As a user I want to use beautiful nice UI minimalist

2. Features:
- Show new jokes when user clicks new jokes
- Add the joke in nice paragraph or text area
- Counter of the jokes user already laughed about
- Set this counter into chrome.local.storage


3. Flowchart 
- async operation(api) for getting the jokes
- handle bad response and network error
- setting listener for the button (eventListener)
- tracking the global variable for joke already laughed about it 
- save the jokes count in the localstorage. so it stays there when user close/or/ refresh the window
